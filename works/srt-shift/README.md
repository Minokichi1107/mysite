# srt-shift

字幕（.srt）のタイミングを調整するための小さなスクリプトです。

## 概要
学習・個人利用を目的に作成しました。  
公開されている情報やサンプルを参考にしつつ、
自分の用途に合わせて調整・理解したものです。

## できること
- 字幕の表示タイミングを秒単位でずらす

## 状態
- 動作確認は個人環境のみ
- 仕様・完成度は今後調整予定

## 注意
- 学習目的のため、保証はありません
- 実行は自己責任でお願いします
